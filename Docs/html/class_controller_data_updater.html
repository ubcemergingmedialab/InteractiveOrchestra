Detailed Description

Updates relevant data being tracked using the controller. This data is displayed and updated dynamically in the inspector and is to be used by team to view the data during screen capture (recording of the gestures) to track gesture points of the user and map to corresponding data.

Parameters
gestureVelocitycurrent velocity of the gesture at a particular data point
accelerationcurrent acceleration of the baton at a particular data point
positionVectorcurrent xyz coordinates of the occulus controller (i.e. baton) in the world space
directionVectorcurrent direction coordinates of the occulus controller (i.e. baton) in the world space
distanceCoveredP1amount of distance (from full trial) that the baton has covered in the P1 region
distanceCoveredP2amount of distance (from full trial) that the baton has covered in the P2 region
timeRelativeToPrepTime runner starting from the beginning of the prep. beat to current time
region"" if current data point is in neither region. Otherwise, either "P1" or "P2".
velocityMagnitudeMagnitude of the velocity
angleToBP1Angle to point BP1 on the plane relative to horizontal plane
